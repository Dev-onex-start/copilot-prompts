# 🤖 Agente Inteligente de Planejamento de Carreira

## 📌 Sobre o Projeto

Essa é a entrega do meu projeto nos aprendizados do curso em parceria da Caixa com a DIO. Aprendi bastante e o enriquecimento dos meus prompts é notável.

O projeto consiste na construção de um agente inteligente capaz de atuar como entrevistador e planejador de carreira, gerando recomendações personalizadas a partir de entradas estruturadas.

A proposta evidencia a evolução na engenharia de prompts, organização de informações e capacidade de transformar interações com IA em soluções práticas.

---

## 🎯 Objetivo

Desenvolver um agente capaz de:

- Simular uma entrevista inicial  
- Coletar dados relevantes do usuário  
- Gerar um plano estruturado de desenvolvimento profissional  
- Organizar informações de forma clara e aplicável  

---

## 🧠 Resultado Gerado pelo Agente

Abaixo está o resultado original gerado a partir da interação com o agente, mantido integralmente como evidência dos aprendizados e da capacidade de estruturação de prompts:

---

> Olá! Recebi suas informações do entrevistador.  
>  
> Vejo que você escolheu **Desenvolvedor Web Full Stack** e tem **16 horas por semana** para estudar. Perfeito!  
>  
> Vou montar agora seu plano completo personalizado:
>  
> ---
>  
> 🧩 **VISÃO DO DIA A DIA**  
>  
> Como é o trabalho de um(a) Desenvolvedor Web Full Stack:  
> - Criar interfaces web responsivas e acessíveis  
> - Desenvolver APIs e serviços de backend  
> - Integrar banco de dados com aplicações  
> - Realizar testes e corrigir bugs  
> - Colaborar com designers, analistas e outros devs em projetos  
>  
> ---
>  
> 🧠 **MAPA DE SKILLS**  
>  
> **CORE SKILLS (essenciais):**  
> - HTML, CSS, JavaScript (frontend)  
> - Node.js e Express (backend)  
> - Banco de dados (SQL e NoSQL)  
>  
> **NICE-TO-HAVE (complementares):**  
> - React ou Angular  
> - Docker e noções de CI/CD  
>  
> **FERRAMENTAS E TECNOLOGIAS:**  
> - Git/GitHub  
> - VS Code  
> - MongoDB / PostgreSQL  
> - APIs REST e GraphQL  
>  
> ---
>  
> 📅 **ROADMAP DE 90 DIAS**  
>  
> **ADAPTADO PARA:** 16h/semana  
>  
> **MÊS 1 - FUNDAMENTOS**  
> - **Semana 1-2:** revisar HTML, CSS e JavaScript; criar pequenos projetos (landing pages).  
> - **Semana 3-4:** aprender Node.js básico e Express; construir primeira API simples.  
>  
> **MÊS 2 - PRÁTICA**  
> - **Semana 5-6:** introdução a React; criar SPA simples consumindo API.  
> - **Semana 7-8:** aprender banco de dados (MongoDB e PostgreSQL); integrar com backend.  
>  
> **MÊS 3 - PORTFÓLIO E PREPARAÇÃO**  
> - **Semana 9-10:** desenvolver projeto completo Full Stack (frontend + backend + DB).  
> - **Semana 11-12:** preparar documentação, publicar no GitHub e simular entrevistas técnicas.  
>  
> ---
>  
> 🚀 **PROJETO DE PORTFÓLIO**  
>  
> **PROJETO:** Plataforma de Gestão de Tarefas com Dashboard  
>  
> **O QUE FAZER:** criar uma aplicação web full stack que permita cadastro de usuários, login e gerenciamento de tarefas com estatísticas.  
>  
> **ENTREGÁVEIS:**  
> - Frontend responsivo em React  
> - API REST em Node.js/Express  
> - Banco de dados conectado (MongoDB ou PostgreSQL)  
>  
> **CRITÉRIOS DE ACEITAÇÃO:**  
> - Usuário consegue se registrar e autenticar  
> - CRUD completo de tarefas  
> - Deploy funcional em Vercel/Render/Heroku  
>  
> **DICA:** comece pelo backend simples e vá evoluindo para o frontend com gráficos e filtros.  
>  
> ---
>  
> 💬 **ROTEIRO DE ENTREVISTAS**  
>  
> **Pergunta 1:** "Fale sobre seu projeto de portfólio"  
> **Como responder:** explique objetivo, tecnologias usadas e desafios superados.  
>  
> **Pergunta 2:** "Qual diferença entre frontend e backend?"  
> **Como responder:** destaque que frontend é interface e experiência do usuário, backend é lógica e dados.  
>  
> **Pergunta 3:** "Como você versiona seu código?"  
> **Como responder:** cite Git/GitHub, branches e pull requests.  
>  
> **Pergunta 4:** "Você já trabalhou em equipe?"  
> **Como responder:** mencione experiências anteriores em vendas e qualidade, mostrando colaboração e comunicação.  
>  
> **Pergunta 5:** "Por que quer ser desenvolvedor Full Stack?"  
> **Como responder:** conecte seu interesse em dados e código com a possibilidade de construir soluções completas.
